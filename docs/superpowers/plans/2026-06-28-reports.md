### Task 1: Report aggregations + writeReportTab
**Files:** Create `packages/worklog-core/src/data/reports.ts` + `reports.test.ts`; export from `index.ts`.

**Interfaces — Produces:**
```ts
hoursByEmployee(att: Attendance[], range: {from,to}): { employeePhone: string; hours: number }[]   // sum closed/corrected hours in range
hoursByLocation(att: Attendance[], instLocById: Map<string,string>, range): { location: string; hours: number }[]
attendanceExceptions(att: Attendance[], instById: Map<string, ShiftInstance>, range): { employeePhone: string; date: string; location: string; kind: 'late'|'out_of_zone' }[]
writeReportTab(gateway, tab: string, header: string[], rows: string[][]): Promise<void>   // writeHeaderRow + appendRow each
```

- [ ] **Step 1: Failing tests** `reports.test.ts`:
```ts
import { test } from 'node:test';
import assert from 'node:assert/strict';
import { createMemoryGateway } from '@scourage/sheets-helper';
import { hoursByEmployee, hoursByLocation, attendanceExceptions, writeReportTab } from './reports.ts';

const att = (over={}) => ({ id:'a', instanceId:'i1', employeePhone:'p1', date:'2026-07-01', checkInAt:'2026-07-01T08:10:00.000Z', checkInLat:'', checkInLng:'', checkInPhoto:'', checkInInGeofence:true, checkOutAt:'2026-07-01T16:00:00.000Z', checkOutLat:'', checkOutLng:'', checkOutPhoto:'', checkOutInGeofence:true, hours:'8', status:'closed', ...over });

test('hoursByEmployee sums closed hours in range', () => {
  const rows = [att({}), att({employeePhone:'p1', hours:'4', date:'2026-07-02'}), att({employeePhone:'p2', hours:'5'}), att({status:'open', hours:'9'})];
  const r = hoursByEmployee(rows as any, { from:'2026-07-01', to:'2026-07-31' });
  assert.deepEqual(r.find(x=>x.employeePhone==='p1'), { employeePhone:'p1', hours:12 });
  assert.deepEqual(r.find(x=>x.employeePhone==='p2'), { employeePhone:'p2', hours:5 });
});
test('hoursByLocation maps instance→location', () => {
  const loc = new Map([['i1','Site A']]);
  const r = hoursByLocation([att({}) ] as any, loc, { from:'2026-07-01', to:'2026-07-31' });
  assert.deepEqual(r, [{ location:'Site A', hours:8 }]);
});
test('attendanceExceptions flags out-of-zone and late', () => {
  const inst = new Map([['i1', { id:'i1', templateId:'t', location:'Site A', date:'2026-07-01', start:'08:00', end:'16:00', headcount:1, status:'scheduled' }]]);
  const late = att({ checkInAt:'2026-07-01T08:20:00.000Z' });            // 20m after 08:00 (grace 15) → late
  const ooz  = att({ checkInInGeofence:false, checkInAt:'2026-07-01T08:00:00.000Z' });
  const ex = attendanceExceptions([late, ooz] as any, inst as any, { from:'2026-07-01', to:'2026-07-31' });
  assert.ok(ex.some(e=>e.kind==='late'));
  assert.ok(ex.some(e=>e.kind==='out_of_zone'));
});
test('writeReportTab writes header + rows to a new tab', async () => {
  const g = createMemoryGateway({});
  await writeReportTab(g, 'Report X', ['a','b'], [['1','2'],['3','4']]);
  const rows = g.dump()['Report X'];
  assert.deepEqual(rows[0], ['a','b']); assert.equal(rows.length, 3);
});
```

- [ ] **Step 2: Run — fail.**
- [ ] **Step 3: Implement `reports.ts`:**
  - `inRange(date, r)` = `date >= r.from && date <= r.to`.
  - `hoursByEmployee`: filter `status` closed/corrected + inRange; sum `Number(hours)||0` grouped by employeePhone; return sorted by phone.
  - `hoursByLocation`: same hours filter; `location = instLocById.get(instanceId) ?? '—'`; sum per location.
  - `attendanceExceptions`: for each closed/corrected att inRange: `out_of_zone` if `!checkInInGeofence || !checkOutInGeofence`; `late` if the instance exists and `Date.parse(checkInAt) > Date.parse(\`${inst.date}T${inst.start}:00Z\`) + 15*60000`. Push one row per kind that applies.
  - `writeReportTab(gateway, tab, header, rows)`: `await gateway.writeHeaderRow(tab, header); for (const r of rows) await gateway.appendRow(tab, r);`
- [ ] **Step 4: Export** the four from `index.ts`.
- [ ] **Step 5: Run — pass + typecheck.**
- [ ] **Step 6: Commit.** `git commit -m "feat(core): report aggregations + writeReportTab"`

### Task 2: `/admin/reports` page + route
**Files:** Create `packages/web/app/admin/reports/page.tsx` + `reports-client.tsx`, `packages/web/app/api/admin/reports/route.ts`; add a reachable Reports link.

- [ ] **Step 1: Route** `POST /api/admin/reports` — `requireAdmin`. Body `{ type: 'hours_employee'|'hours_location'|'payroll'|'exceptions', from, to }`. Load the data needed (`listAttendance({from,to})`, `listWorkers`, `listInstances({from,to})`, and for payroll also templates/places/adjustments). Compute the chosen report into `{ header: string[], rows: string[][] }`:
  - hours_employee: header `['Worker','Hours']`; from `hoursByEmployee` mapped phone→name.
  - hours_location: header `['Location','Hours']`; from `hoursByLocation`.
  - payroll: header `['Worker','Structure','Hours','Gross','Bonuses','Penalties','Net']`; reuse the payroll computation (per active worker: build items via `resolveHourlyRate` + `computePay`, same as `/admin/payroll/page.tsx` — copy that logic).
  - exceptions: header `['Worker','Date','Location','Issue']`; from `attendanceExceptions` mapped phone→name.
  Then `const tab = \`Report ${type} ${from}..${to}\`.slice(0,90); await writeReportTab(getGateway(), tab, header, rows);` Return `{ ok:true, tab, header, rows }`. Import depth `../../../../lib`.

Handle errors.
- [ ] **Step 4: Reachable** — add a **Reports** link (admin-nav tab OR Payroll page header). MUST be reachable.
